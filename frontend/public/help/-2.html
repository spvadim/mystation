Новый куб

 Начало  Предыдущий  Следующий

При необходимости собрать Куб вручную, вводя данные каждой пачки, необходимо нажать на кнопку “Новый Куб”.

 

Порядок работы:

1.Нажмите на кнопку "Новый куб".

 

При нажатии кнопки “Новый Куб”, откроется окно создания нового Куба.

 

2. Заполните форму.

1.Состав Куба.

2.Дата партии, к которой будет принадлежать новый Куб.

3.Добавить, с помощью мобильного сканера штрих-кода, в состав Куба пачки, которые будут принадлежать Кубу.

4.Добавить, с помощью мобильного сканера штрих-кода, QR-код нового Куба.
