Сброс пачек за пинцет

 Начало  Предыдущий  Следующий

При попадании неучтенной пачки под пинцет, система сбросит две пачки из-под пинцета.

 

Порядок работы при обнаружения сброса пачек за пинцет:

1.Снять с линии сброшенные пачки.

2.Физически удалить (оторвать) этикетки с QR-кодом с обеих сброшенных пачек.

3.Пачки, уже без этикеток, вернуть на линию перед аппликатором, где будет присвоен новый QR-код.

 

 

Без смены QR-кода, - пачки будут постоянно сбрасываться с производственной линии.

 

 

 

1.Нельзя руками вытаскивать пачки уже прошедшие в пинцет.

2.Нельзя вручную проталкивать пачки в пинцет.

3.Если будет зафиксировано ручное вмешательство, то Куб собираемый в этот момент будет окрашен в красный цвет и его нужно будет пересобирать вручную.
